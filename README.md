# ❤️ Heart Stroke Prediction System
## Project Overview

The Heart Stroke Prediction System is a Machine Learning application that predicts whether a person is at risk of heart disease based on medical information provided by the user.

The model was trained using the K-Nearest Neighbors (KNN) algorithm and deployed using Streamlit.

## Features

- Predicts heart disease risk
- User-friendly Streamlit interface
- Real-time prediction
- Data preprocessing
- Feature scaling
- Machine Learning model deployment

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib
- Streamlit

## Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Feature Scaling
6. Model Training
7. Model Evaluation
8. Model Saving
9. Streamlit Deployment

## Algorithm

K-Nearest Neighbors (KNN)


## Dataset

The dataset contains medical attributes such as:

- Age
- Sex
- Chest Pain Type
- Cholesterol
- Blood Pressure
- Maximum Heart Rate
- Exercise Angina
- ST Depression

Target:

0 → No Heart Disease

1 → Heart Disease

## Model Performance

Training Accuracy : 85%

Testing Accuracy : 86%
